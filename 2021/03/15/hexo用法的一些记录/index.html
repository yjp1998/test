hexo用法的一些记录

发表于 2021-03-15

记录hexo的一些用法

创建文章指令

hexo new ‘文章名’ 创建后会在 source_posts 目录下
文章的后缀名是以md命名的文件格式，遵循markdown语法

分类及标签添加

新建一篇文章默认是没有分类及标签的，可以手动进行修改。然后文章就会自动分类以及加上对应的标签了。
tags: 标签名
categories: 分类名

文章过长，使用阅读全文分屏

比较简单的方式：只需要在文章内容需要分割的地方加一句命令即可
<!-- more -->

markdown 语法

一个#是一级标题，以此类推。支持六级标题

字体

加粗：要加粗的文字左右分别用两个号包起来
斜体：要倾斜的文字左右分别用一个号包起来
斜体加粗：要倾斜和加粗的文字左右分别用三个*号包起来
删除线：要加删除线的文字左右分别用两个~~号包起来

引用

在引用的文字前加>即可

分割线

三个或三个以上的 - 或者 * 都可以

超链接

[超链接名](超链接地址 “超链接title”)title可加可不加

图片插件使用

安装图片插件
npm install hexo-asset-image –save
在站点_config.yml配置，修改 post_asset_folder: true
新建文章后，会出现文章同名的文件夹，用来存放照片
然后就可以使用超链接的语法来引用

代码

单行代码：代码之间分别用一个反引号包起来
代码块：分别用三个反引号包起来，且两边的反引号单独占一行
